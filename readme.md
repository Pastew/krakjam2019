# Krak Jam 2019
Subject: What home means to you?

Game: Second thought

## Controls
* Player 1
    * Moving: Gamepad 1 analog stick / WSAD
    * Action: Gamepad 1: A / Keyboard: Z

* Player 2
    * Moving: Gamepad 2 analog stick / IJKL
    * Action: Gamepad 2: A / Keyboard: M
